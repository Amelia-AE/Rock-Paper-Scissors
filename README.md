# Rock-Paper-Scissors
Zuri project

If the two players choose the same “character” it’s a tie, and the game repeats itself.
Rock beats Scissors,
Paper beats Rock,
Scissors beats Paper.

Possible Options:
"R" for "rock", 
"P" for "paper", 
"S" for "scissors".

The game is played until the best 3 of five wins.
Winner is declared and the game ends.
